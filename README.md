This is a simple python script used to compose and resize PNG images.

The exact use and setup is explained in the following blog post
[Python Libraries: Opening and rescaling images with Pillow](https://michaehrig.wixsite.com/mics-code-corner/post/python-libraries-opening-and-rescaling-images-with-pillow)

In short three PNG per product variant are taken, composed and then converted to JPEG files.
